El atributo "rel" es para especificar que relación tiene el link con la página.
rel=alternate, author, bookmark, external, help, license, next, nofollow, noreferrer, noopener, prev, search, tag.

More about rel

El atributo "target" es para indicar donde abrirá el link.
target=_blank, _parent, _self, _top, framename.
Y tambien indica para que tipo de dispositivos está optimizado el link con el atributo "media".
